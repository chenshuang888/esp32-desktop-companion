## 阅读建议

- **想用这个项目** → 只看 `参考/` 5 篇
- **想理解这个项目** → 加看 `架构/项目深度分析报告.md` 和 `架构/三层架构图.md`
- **想看演进过程** → 按 `日志/` 文件名升序读，每篇都是一次具体决策的来龙去脉
- **想找某个具体决策的原因** → 用 GitHub 的全文搜索，比目录浏览快
